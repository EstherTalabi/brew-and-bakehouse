# Brew & Bakehouse
By Esther Talabi

A web-based application for a coffee shop and bakery.
>## Technologies Used
* HTML
* CSS
* Bootstrap
* JavaScript
* Git

>## Description
This web-based application is designed for **Brew & Bakehouse**, a coffee shop and bakery located in Yaba, Lagos. It enables customers to browse the full menu, filter products by category, add items to their shopping cart, and submit pickup orders by entering their details.



>## Setup/Installation Requirements
* Clone this repository to your desktop
* Navigate to the top level of the directory
* Click on index.html to get started
